# MEAN Todo App

To run the application you need MongoDB, Node and NPM installed.

Clone this repository and type:

```
npm install
```

In server.js edit line:

```
var dbUrl = 'mongodb://localhost/taskbase';
```
to your mongodb url. Then run:

```
npm start
npm run server
```
